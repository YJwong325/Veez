# Veez
A web app to showcase my favourite animated works and media.

## Technologies used
### Frontend
- HTML
- Tailwind CSS + DaisyUI
- EJS (templating engine)

### Backend
- Node.js
- Express.js
- JavaScript
- Fetch API + GraphQL

## Disclaimer
- This project uses the [AniList API](https://docs.anilist.co/) to fetch anime and manga data via GraphQL queries.
- I do not claim ownership of any of the data used in this project.
- All data used in this project belong to their original creators and authors.
